## Technical Approach

保留 TS 版五层架构思想，用 Rust 惯用方式重新表达每层：

| 层 | TS 实现 | Rust 实现 |
|---|---------|----------|
| Host | 手动装配 world/eventBus/executor | `alice` binary crate，装配各 lib crate 组件 |
| Core | 6 个 class（World, EventBus, EffectExecutor, SystemRegistry, ToolScheduler, AbortManager） | 对应 struct + trait，World 泛型化 |
| Business | 5 个 const System 函数 | 5 个函数，实现 `System` trait 或函数指针 |
| Data | 5 个 Component factory 函数 + 1 个 ComponentDataMap interface | 泛型 trait `World<Components>`，编译期确定组件集合 |
| Infra | StreamingProvider interface + MiddlewarePipeline class | `StreamingProvider` trait + `Middleware` trait |

## Key Type Designs

### World (泛型替代动态 ECS)

```rust
// 编译期确定的组件集合，零运行时 downcast
pub trait World {
    type Components;
    fn get<C: Component>(&self) -> &C;
    fn get_mut<C: Component>(&mut self) -> &mut C;
    fn snapshot(&self) -> Snapshot<Self::Components>;
}
```

### System (纯函数 trait)

```rust
pub trait System<W: World> {
    fn process(&self, snapshot: &Snapshot<W::Components>, event: &Event) -> Vec<Effect>;
}
// 同时支持 Fn(&Snapshot, &Event) -> Vec<Effect> 函数指针
```

### Event (枚举替代 discriminated union)

```rust
pub enum Event {
    Input(InputEvent),
    LLMStream(LLMStreamEvent),
    Tool(ToolEvent),
    System(SystemEvent),
}
```

### Effect (枚举 + EffectExecutor)

```rust
pub enum Effect {
    CallLLM { messages: Vec<Message> },
    ExecuteTool { name: String, args: serde_json::Value },
    AppendMessage { message: Message },
    UpdateComponent { /* generics 处理 */ },
    Emit { event: Event },
    Render { content: String, stream: StreamType },
    Abort { reason: String },
}
```

### 流式接口

```rust
// Provider 的流式输出通过 tokio::sync::mpsc 或 async_stream
pub trait StreamingProvider {
    fn stream_chat(
        &self,
        messages: Vec<Message>,
    ) -> impl Stream<Item = Result<LLMStreamEvent, Error>>;
}
```

## Dependencies (minimal)

**alice-core:** `tokio`, `serde`/`serde_json`, `thiserror`
**alice-providers:** `alice-core`, `reqwest` (HTTP client)
**alice-tools:** `alice-core`, `serde_json`
**alice (bin):** 所有三个 lib crate, `anyhow`, `tokio`
