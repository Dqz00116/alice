- [ ] **Step 1: Refactor SSE loop to track tool-call state**

  In `stream_chat`, replace the simple `parse_sse_data` dispatch with inline event handling. Maintain local variables:

  ```rust
  let mut current_tool: Option<(String, String, String)> = None; // (id, name, partial_json)
  ```

  For each SSE data line parse JSON and match `type`:

  - `content_block_start` with `content_block.type == "tool_use"`:
    ```rust
    current_tool = Some((id, name, String::new()));
    ```
  - `content_block_delta` with `delta.partial_json`:
    ```rust
    if let Some((_, _, ref mut partial)) = current_tool {
        partial.push_str(partial_json);
    }
    ```
  - `content_block_stop`:
    ```rust
    if let Some((id, name, partial)) = current_tool.take() {
        let input: serde_json::Value = serde_json::from_str(&partial).unwrap_or(serde_json::json!({}));
        yield LLMStreamEvent::ToolCall {
            tool_call: ToolCall {
                id,
                call_type: "tool_use".into(),
                function: FunctionCall { name, arguments: input.to_string() },
            },
        };
    }
    ```
  - Keep existing `text`/`thinking` delta handling.
  - Keep `message_stop` and `[DONE]` → `StreamEnd`.

### Task 2: Fix Anthropic Message Serialization

**Files:**
- Modify: `crates/alice-providers/src/anthropic.rs`
- Modify: `crates/alice-providers/tests/anthropic_test.rs`
- Modify: `crates/alice-providers/tests/anthropic_live_test.rs` (if needed)

- [ ] **Step 1: Update `Message::Assistant` serialization**

  ```rust
  Message::Assistant { content, tool_calls } => {
      if tool_calls.is_empty() {
          serde_json::json!({ "role": "assistant", "content": content })
      } else {
              let mut content_blocks = vec![serde_json::json!({ "type": "text", "text": content })];
              for tc in tool_calls {
                  let input: serde_json::Value = serde_json::from_str(&tc.function.arguments)
                      .unwrap_or(serde_json::json!({}));
                  content_blocks.push(serde_json::json!({
                      "type": "tool_use",
                      "id": tc.id,
                      "name": tc.function.name,
                      "input": input,
                  }));
              }
              serde_json::json!({ "role": "assistant", "content": content_blocks })
      }
  }
  ```

- [ ] **Step 2: Update `Message::Tool` serialization**

  ```rust
  Message::Tool { content, tool_call_id } => {
      serde_json::json!({
          "role": "user",
          "content": [{
              "type": "tool_result",
              "tool_use_id": tool_call_id,
              "content": content,
          }]
      })
  }
  ```
